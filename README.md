# Interpolation Search Project

## Objective
To implement and analyze Interpolation Search on uniformly distributed arrays.

## Features
- Generates a sorted uniform array
- Searches using Interpolation Search
- Counts comparisons
- Measures execution time
- Displays complexity analysis
- Plots performance graph

## Time Complexity
Best Case: O(1)

Average Case: O(log log n)

Worst Case: O(n)

## Space Complexity
O(1)
